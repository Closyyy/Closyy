🔧 Configura Firebase primero
Abre este index.html con cualquier editor de texto y rellena tus datos de Firebase donde pone TU_API_KEY etc.
Paso 1 — Crear proyecto Firebase
Ve a console.firebase.google.com → "Crear proyecto" → ponle un nombre → continuar
Paso 2 — Obtener tu configuración
En tu proyecto → ⚙️ Configuración → pestaña "General" → baja hasta "Tus apps" → añade app Web (</>) → copia el objeto firebaseConfig
Paso 3 — Activar servicios
• Authentication → Comenzar → Proveedores: activa Correo/contraseña y Google
• Firestore Database → Crear base de datos → Modo prueba (30 días) → elegir región
• Storage → Comenzar → Modo prueba
Paso 4 — Pegar tus datos aquí
Busca en el HTML la sección marcada con 🔧 y reemplaza los 6 valores: apiKey, authDomain, projectId, storageBucket, messagingSenderId, appId
Paso 5 — API Key de Gemini (gratis, para el botón IA)
Ve a aistudio.google.com → Get API Key → Create API key → copia la clave y pégala donde pone TU_GEMINI_API_KEY
Paso 6 — Reglas de Firestore
Firestore → Reglas → pega esto y publica:
rules_version='2'; service cloud.firestore { match /databases/{db}/documents { match /users/{uid}/{doc=**} { allow read,write: if request.auth.uid==uid; } } }
